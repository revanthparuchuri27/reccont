# 🎬 Movie Recommendation System

A full-stack web application that recommends movies based on genre.

## Features
- Genre-based movie recommendations
- Movie posters and ratings
- Trending movies section
- Responsive UI

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Dataset: TMDB Movies Dataset

## How to Run

1. Install dependencies:
pip install flask flask-cors pandas requests

2. Run server:
python app.py

3. Open frontend:
frontend/index.html
